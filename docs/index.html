Architecture

Simple flow, clear layers, easy to deploy.

01 Request masuk

Query, user id, atau activity event diterima lewat endpoint FastAPI.

02 Signal mixing

Model content, collaborative, dan trending dirangkai jadi satu skor gabungan.

03 Result delivery

Daftar rekomendasi dikirim balik dengan metadata yang siap dipakai frontend.

API

Endpoint inti yang paling penting.

GET /healthHealth check server.
GET /search/semanticSemantic content search.
GET /recommendations/trendingTrending books.
POST /recommendations/chatChat-based recommendation.
POST /reindexTrigger model rebuild.
POST /reload-catalogReload catalog data.
